REFRESH INCREMENTAL MATERIALIZED VIEW REFRESH INCREMENTAL MATERIALIZED VIEW 功能描述 REFRESH INCREMENTAL MATERIALIZED VIEW会以增量刷新的方式对物化视图进行刷新。 注意事项 增量刷新仅支持增量物化视图。 刷新物化视图需要当前用户拥有基表的SELECT权限。 语法格式 REFRESH INCREMENTAL MATERIALIZED VIEW mv_name; 参数说明 mv_name 要刷新的物化视图的名称。 示例 --创建一个普通表。 gaussdb=# CREATE TABLE my_table (c1 int, c2 int)
WITH(STORAGE_TYPE=ASTORE);

--创建增量物化视图。 gaussdb=# CREATE INCREMENTAL MATERIALIZED VIEW my_imv AS SELECT * FROM my_table;

--基表写入数据。 gaussdb=# INSERT INTO my_table VALUES(1,1),(2,2);

--对增量物化视图my_imv进行增量刷新。 gaussdb=# REFRESH INCREMENTAL MATERIALIZED VIEW my_imv;

--删除增量物化视图。 gaussdb=# DROP MATERIALIZED VIEW my_imv;
