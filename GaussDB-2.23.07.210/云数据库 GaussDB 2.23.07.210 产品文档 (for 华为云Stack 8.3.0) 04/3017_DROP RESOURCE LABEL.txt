DROP RESOURCE LABEL DROP RESOURCE LABEL 功能描述 删除资源标签。 注意事项 只有poladmin，sysadmin或初始用户才能执行此操作。 语法格式 1 DROP RESOURCE LABEL [ IF EXISTS ] label_name [, ...]; 参数说明 label_name 资源标签名称。 取值范围：字符串，要符合 标识符命名规范 。 示例 请参见CREATE RESOURCE LABEL的 8.12.8.37-示例 相关链接 ALTER RESOURCE LABEL ， CREATE RESOURCE LABEL 。 父主题： D 版权所有 © 华为技术有限公司
