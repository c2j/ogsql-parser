seq_query seq_query 功能介绍 seq_query用于在数据库实例中升级过程中，从etcd读取sequence的数据或将sequence数据插入etcd。解决sequence特殊字符无法通过etcdctl工具读取的问题，以及sequence的increment为负数无法使用etcdctl工具插入etcd的问题。 参数说明 seq_query参数可以分为两类： option参数，参见 表1 。 公共参数，参见 表2 。 表1 option参数 参数 参数说明 get 从etcd获取sequence相关数据。 set 将seqkey替换uuid后的sequence插入etcd。 表2 公共参数 参数 参数说明 -p ETCDIP:ETCDPORT 所有的etcd的ip和port，ip和port使用: 分割，etcd之间使用，分割。 -f SEQUENCE_FILE sequence相关信息的文件绝对路径。 -v 特殊字符转义，添加-v表示需要转义。 父主题： 系统内部调用的工具 版权所有 © 华为技术有限公司
