生成登录脚本 生成登录脚本 DBS运维管理平台 提供运维紧急处理通道，可以通过该功能生成登录节点的脚本信息登录数据库后台节点。登录节点脚本生成后，您可以： 登录后台节点。 将数据库文件下载到后台节点。 从后台服务节点上传文件到数据库。 操作步骤 登录DBS运维管理平台 。 选择 “实例运维 > 实例管理 > 实例列表” 。 选择需要登录的实例信息，单击右侧“操作”列的 “生成登录脚本” 。 在弹窗中输入或者选择多个节点ID，确认对应的节点名称和类型，单击 “确定” 。 图1 生成登录脚本 根据返回提示，获取脚本生成的节点IP和路径。 例如： generate script or submit script job success, ops-admin ip:{hostIp}/tmp/opsTool/{instanceId} 需要登录到ops-admin所在的后台服务节点hostIp，在指定路径“/tmp/opsTool/{instanceId}”下，使用脚本即可。 登录脚本有效期约2小时，过期后自动删除并重置该节点OS密码信息。 在弹窗中输入或者选择多个节点ID，确认对应的节点名称和类型，单击 “显示命令” ，获取命令示例或者单击 “下载密码” ，获取密码信息（包含命令示例）。 图2 生成登录脚本 Mike密码：数据库节点的Mike账号SSH私钥的密码。 Root密码：数据库节点的Root账号的密码。 Ruby密码：数据库节点的Ruby账号的密码。 脚本使用方法如下： 使用用户opsadmin登录后台服务节点，节点地址为 5 中获取的hostIp。 默认密码请参考《华为云Stack 8.5.0 账户一览表》中 “数据库” 页签。选择“云服务”为“DBSPlatform”、“账户所在的管理虚拟机/节点”为“DBS-OpsAdmin”，获取 “opsadmin” 账号名对应的密码。 为了您的账户安全，请定期修改密码。 执行以下命令，切换到root。 sudo su - root 默认密码请参考《华为云Stack 8.5.0 账户一览表》中 “数据库” 页签。选择“云服务”为“DBSPlatform”、“账户所在的管理虚拟机/节点”为“DBS-OpsAdmin”，获取 “root” 账号名对应的密码。 为了您的账户安全，请定期修改密码。 执行如下指令，切换到service用户。 su - service 进入到脚本目录。 cd /tmp/opsTool/ {instanceId} 登录到节点命令。 登录到节点的命令为 6 中获得的命令。 例如： sh connNode.sh connect {nodeId} {mngPPh} {rtPd}

{mngPPh}是步骤6中“下载密码”得到的文档里的“Mike密码”的值。
{rtPd}是步骤6中“下载密码”得到的文档里的“Root密码”的值。 引申功能： 从节点拉取文件（将需要拉取的文件复制到/home/Mike目录下，文件权限修改为Mike）： sh connNode.sh ${nodeId} pull /home/Mike / { filename } {local_directory} {mngPPh} 从本地推送文件（本地推送的文件及其所在的目录权限为service）： sh connNode.sh ${nodeId} push {local_directory}/{filename} /home/Mike {mngPPh} 其中，{mngPPh}是 6 中“下载密码”得到的文档里的“Mike密码”的值。 父主题： 实例管理 版权所有 © 华为技术有限公司
