ALTER GLOBAL CONFIGURATION ALTER GLOBAL CONFIGURATION 功能描述 新增、修改系统表gs_global_config的key-value值。如果修改的参数已经存在，则修改；如果不存在则新增。 注意事项 仅支持数据库初始用户运行此命令。 参数名称不能为weak_password、undostoragetype。 语法格式 ALTER GLOBAL CONFIGURATION with(name=value, name=value...); 参数说明 name 参数名称，text类型，不能为weak_password、undostoragetype，除此之外没有限制。 value 参数值，text类型。 示例 1
