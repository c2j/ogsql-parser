EXPDP TABLE EXPDP TABLE 功能描述 导出与表相关的索引、sequence、分区、toast、toast index等全部文件。 语法格式 EXPDP TABLE table_name LOCATION = 'directory'; 参数说明 table_name 导出的表名。 directory 导出文件的存储目录。 示例 --EXPDP TABLE语法用于细粒度备份恢复，由备份恢复工具调用，用户直接调用可能会提示目录不存在等报错，不推荐用户直接调用该SQL。 gaussdb=# EXPDP TABLE test_t LOCATION = '/data1/expdp/table0'; 父主题： E 版权所有 © 华为技术有限公司
