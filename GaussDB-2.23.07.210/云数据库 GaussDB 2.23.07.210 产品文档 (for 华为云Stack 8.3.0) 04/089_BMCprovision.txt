BMC网段与provision平面网络互通 BMC网段与provision平面网络互通 目的 检查BMC网段与provision平面网络是否打通。 前提条件 底座已安装。 操作步骤 登录FusionSphere OpenStack安装部署界面（CPS）首节点后台。 从底座LLD "Protal" 页签，获取FusionSphere OpenStack安装部署界面（CPS）的IP，从后台登录 密码从账户一览表中“A类（后台）”页签，FusionSphere OpenStack对应的FSP账号密码， 登录后切换root用户。 su - root 执行以下命令获取分配的provision ip。 ip a | grep provision 登入ServiceOM 选择 裸金属 > 待发放BMC IP。 执行以下命令。 ping –I { provision ip} { bmc ip } 预期结果 无丢包现象且延时低，网络互通。 父主题： 检查BMS形态依赖项 版权所有 © 华为技术有限公司
