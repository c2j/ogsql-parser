GLOBAL_STAT_USER_FUNCTIONS GLOBAL_STAT_USER_FUNCTIONS 提供 整个集群 中各个节点的用户所创建的函数的状态的统计信息。 表1 GLOBAL_STAT_USER_FUNCTIONS字段 名称 类型 描述 node_name name 函数的OID。 funcid oid 函数的id。 schemaname name 此函数所在模式的名称。 funcname name 用户function的名称。 calls bigint 函数被调用的次数。 total_time double precision 调用此function的总时间花费，包含调用其它function的时间(单位：毫秒)。 self_time double precision 调用此function自己时间的花费，不包含调用其它function的时间(单位：毫秒)。 父主题： Object 版权所有 © 华为技术有限公司
