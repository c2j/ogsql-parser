对列表分区表分割分区 对列表分区表分割分区 使用ALTER TABLE SPLIT PARTITION可以对列表分区表分割分区。 例如，假设列表分区表list_sales的分区channel2定义范围为('6', '7', '8', '9')。可以指定分割点('6', '7'）将分区channel2分割为两个分区，并更新Global索引。 ALTER TABLE list_sales SPLIT PARTITION channel2 VALUES ('6', '7') INTO
(
    PARTITION channel2_1, --第一个分区范围是('6', '7'）
    PARTITION channel2_2  --第二个分区范围是('8', '9'）
) UPDATE GLOBAL INDEX; 或者，不指定分割点，将分区channel2分割为多个分区，并更新Global索引。 ALTER TABLE list_sales SPLIT PARTITION channel2 INTO
(
    PARTITION channel2_1 VALUES ('6'),
    PARTITION channel2_2 VALUES ('8'),
    PARTITION channel2_3 --第三个分区范围是('7', '9'）
)UPDATE GLOBAL INDEX; 又或者，通过指定分区值而不是指定分区名来分割分区。 ALTER TABLE list_sales SPLIT PARTITION FOR ('6') VALUES ('6', '7') INTO
(
    PARTITION channel2_1, --第一个分区范围是('6', '7'）
    PARTITION channel2_2  --第二个分区范围是('8', '9'）
) UPDATE GLOBAL INDEX; 若对DEFAULT分区进行分割，前面几个分区不能申明DEFAULT范围，最后一个分区会继承DEFAULT分区范围。 父主题： 分割分区 版权所有 © 华为技术有限公司
