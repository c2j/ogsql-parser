DB_IND_COLUMNS DB_IND_COLUMNS DB_IND_COLUMNS视图显示当前用户可访问的所有索引的字段信息。该视图同时存在于PG_CATALOG和SYS Schema下。 表1 DB_IND_COLUMNS字段 名称 类型 描述 index_owner character varying(64) 索引的所有者。 index_name character varying(64) 索引名。 table_owner character varying(64) 表的所有者。 table_name character varying(64) 表名。 column_name name 列名。 column_position smallint 索引中列的位置。 column_length numeric 列的长度。 char_length numeric 列的长度（字符类型）。 descend character varying(4) 表示列的排序方式：降序（DESC）、升序（ASC）。 collated_column_id numeric 此列提供语言排序的列的内部序列号。 父主题： 其他系统视图 版权所有 © 华为技术有限公司
