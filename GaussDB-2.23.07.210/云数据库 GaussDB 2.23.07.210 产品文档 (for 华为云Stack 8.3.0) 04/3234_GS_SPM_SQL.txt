GS_SPM_SQL GS_SPM_SQL GS_SPM_SQL系统表是用于存储SPM SQL相关信息的系统表，具备sysadmin权限的用户可以对该系统表进行读操作，但只有初始用户才可以对该系统表进行写操作。 表1 GS_SPM_SQL字段 名称 类型 描述 sql_namespace oid schema oid。 sql_hash bigint SPM中SQL的唯一标识。 sql_text text SQL的文本串。 param_num integer SQL执行所需参数的数量。 unique_sql_id bigint GaussDB中SQL的唯一标识。 user oid 登录用户的oid。 create_time timestamp 本记录的创建时间。 父主题： SPM计划管理 版权所有 © 华为技术有限公司
