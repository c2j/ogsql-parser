ALTER WORKLOAD GROUP ALTER WORKLOAD GROUP 功能描述 修改一个负载组，设置并发数量。 注意事项 只要用户对当前数据库有ALTER权限，就可以修改负载组。 不声明ACT_STATEMENTS时，表示不限制负载组所在资源池的并发数量。 语法格式 1
2 ALTER WORKLOAD GROUP wg_name USING RESOURCE POOL pool_name [ WITH ( ACT_STATEMENTS = count ) ]; 参数说明 wg_name 负载组名称。 负载组名称不能和当前数据库里其他负载组重名。 取值范围：字符串，要符合 标识符命名规范 。 pool_name 资源池名称。 取值范围：字符串，已创建的资源池。 count 负载组所在资源池内的并发数量。 取值范围：整型，取值范围为-1 ~ 2147483647‬。缺省值为-1。-1表示不限制数量。 示例 1
