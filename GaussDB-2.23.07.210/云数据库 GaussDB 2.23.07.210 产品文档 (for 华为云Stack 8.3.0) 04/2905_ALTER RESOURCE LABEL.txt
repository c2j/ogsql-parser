ALTER RESOURCE LABEL ALTER RESOURCE LABEL 功能描述 修改资源标签。 注意事项 只有poladmin, sysadmin或初始用户才能执行此操作。 语法格式 1
2 ALTER RESOURCE LABEL label_name ( ADD | REMOVE ) label_item_list [, ...]; label_item_list： 1 resource_type ( resource_path [, ...]) resource_type： TABLE | COLUMN | SCHEMA | VIEW | FUNCTION 参数说明 label_name 资源标签名称。 取值范围：字符串，要符合 标识符命名规范 。 resource_type 指的是要标记的数据库资源类型。 resource_path 指的是描述具体的数据库资源的路径。 示例 1
