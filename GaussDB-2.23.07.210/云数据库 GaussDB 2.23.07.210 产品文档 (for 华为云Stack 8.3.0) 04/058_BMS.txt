BMS高速网络平面两中心间带宽（两地三中心检查） BMS高速网络平面两中心间带宽（两地三中心检查） 目的 BMS高速网络两中心间带宽80Gb。 前提条件 两中心各申请三台BMS实例。 各实例挂载一个在同一高速网络平面的网卡。 操作步骤 租户用户登录ManageOne运营面。 进入“ > 计算 > 裸金属服务器”页面，单击“申请裸金属服务器”，选择对应高速平面。 其中一中心，3台BMS通过高速平面同时分别打流到另一中心的3台BMS，网络侧观察两中心间高速网络带宽。 打流操作： 1. 下载对应版本的iPerf工具，并且在需要测试的服务器上解压（测试的都需要）。 2. 在目的端服务器执行以下命令： cd {path} iperf3 ip {port} -s {path} 为工具在服务器上解压后的文件目录 {port} 为监听端口，一般linux为22，注意防火墙设置，需提供一个已经放通的端口 3. 在源端服务器执行以下命令 cd {path} iperf3 -c {ip} -p {port} -t {time} {ip} 目的端IP {port} 目的端端口 {time} 测试时间 4. iperf客户端连接到iperf服务端成功后会进行带宽（Bandwidth）测试，测试结束后查看结果即可 预期结果 网络侧观察两中心间高速网络带宽约为80Gb。 父主题： BMS实例发放前检查 版权所有 © 华为技术有限公司
