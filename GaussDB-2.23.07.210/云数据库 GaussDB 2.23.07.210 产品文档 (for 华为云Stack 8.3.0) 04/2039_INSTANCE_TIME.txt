INSTANCE_TIME INSTANCE_TIME 提供当前集群节点下的各种时间消耗信息，主要分为以下类型： DB_TIME：作业在多核下的有效时间花费。 CPU_TIME：CPU时间的消耗。 EXECUTION_TIME：执行器内花费的时间。 PARSE_TIME：SQL解析的时间花费。 PLAN_TIME：生成Plan的时间花费。 REWRITE_TIME：SQL重写的时间消耗。 PL_EXECUTION_TIME ：PL/SQL（存储过程）的执行时间。 PL_COMPILATION_TIME：PL/SQL（存储过程）编译时间。 NET_SEND_TIME：网络上的时间花销。 DATA_IO_TIME：I/O时间上的花销。 表1 INSTANCE_TIME字段 名称 类型 描述 stat_id integer 统计编号。 stat_name text 类型名称。 value bigint 时间值（单位：微秒）。 父主题： Instance 版权所有 © 华为技术有限公司
