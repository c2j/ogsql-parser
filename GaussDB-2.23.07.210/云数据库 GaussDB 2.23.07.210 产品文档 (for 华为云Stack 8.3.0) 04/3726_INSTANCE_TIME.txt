INSTANCE_TIME INSTANCE_TIME 提供当前数据库节点下的各种时间消耗信息，主要分为以下类型： DB_TIME：作业在多核下的有效时间花销。 CPU_TIME：CPU的时间花销。 EXECUTION_TIME：执行器内的时间花销。 PARSE_TIME：SQL解析的时间花销。 PLAN_TIME：生成Plan的时间花销。 REWRITE_TIME：SQL重写的时间花销。 PL_EXECUTION_TIME ：PL/SQL（存储过程）执行的时间花销。 PL_COMPILATION_TIME：PL/SQL（存储过程）编译的时间花销。 NET_SEND_TIME：网络上的时间花销。 DATA_IO_TIME：I/O上的时间花销。 表1 INSTANCE_TIME字段 名称 类型 描述 stat_id integer 统计编号。 stat_name text 类型名称。 value bigint 时间值（单位：微秒）。 父主题： Instance 版权所有 © 华为技术有限公司
