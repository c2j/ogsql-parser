登录数据库 登录数据库 操作步骤 登录ManageOne运维面。 在页面的“常用链接”导航栏中，单击“Service OM”，选择区域后进入Service OM界面。 选择“服务列表 > 资源 > 计算资源”，单击“虚拟机”。 在搜索框中输入微服务的节点名称，查询相应管理节点IP。 根据IP使用opsadmin登录。登录完成后，切换root权限。切换至dbadmin用户，执行如下登录命令。 表1 登录命令对应表 微服务 节点名称 登录命令 管理面备份数据库 DBS-GaussBackup gsql -p port -q core -U core-W pwd 默认端口8635 管理面workflow数据库 DBS-GaussBase gsql -p port -q wfs -U wfs -W pwd 默认端口8636 管理面core数据库 DBS-GaussCore gsql -p port -q core -U core-W pwd 默认端口8635 管理面计量、指标数据库 DBS-GaussMetering gsql -p port -q metering -U metering -W pwd gsql -p port -q metric -U metric -W pwd 默认端口8635 port是指数据库端口，pwd是指数据库的用户密码。 用户有可能会变，具体看数据库配置。 用户的默认密码请参考《华为云Stack 8.3.0 账户一览表》中 “数据库” 页签，云服务选择 “DBSPlatform” ，查看对应的账户名称获取密码。 查询数据完毕后，退出数据库。 执行命令： \q 即可退出数据库。 父主题： 操作参考 版权所有 © 华为技术有限公司
