创建数据库用户 创建数据库用户 默认只有集群安装时创建的管理员用户可以访问初始数据库，您还可以创建其他数据库用户帐号。 1 gaussdb =# CREATE USER joe WITH PASSWORD "********" ; 当结果显示为如下信息，则表示创建成功。 1 CREATE ROLE 如上创建了一个用户名为joe，密码为********的用户。如下命令为设置joe用户为系统管理员。 gaussdb=# GRANT ALL PRIVILEGES TO joe; 使用GRANT命令进行相关权限设置，具体操作请参考 GRANT 。 关于数据库用户的更多信息请参考 用户及权限 。 父主题： 操作数据库 版权所有 © 华为技术有限公司
