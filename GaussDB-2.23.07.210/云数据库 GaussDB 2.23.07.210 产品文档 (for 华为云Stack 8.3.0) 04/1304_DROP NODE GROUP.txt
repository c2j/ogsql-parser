DROP NODE GROUP DROP NODE GROUP 功能描述 删除节点组。 注意事项 DROP NODE GROUP是集群管理工具封装的接口，用来实现集群管理。 只有系统管理员或者被授予了节点组DROP权限的用户才能执行该操作。 语法格式 1 DROP NODE GROUP groupname [ DISTRIBUTE FROM src_group_name ]; 参数说明 groupname 要删除的节点组名。 取值范围：已存在的节点组。 示例 请参见CREATE NODE GROUP的 示例 。 相关链接 ALTER NODE GROUP ， CREATE NODE GROUP 父主题： D 版权所有 © 华为技术有限公司
